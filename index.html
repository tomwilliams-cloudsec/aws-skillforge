Tips I Wish I Had Earlier

Use tags religiously. Tag every resource you create in the Free Tier — it's how you track costs and avoid surprise bills.

Set a billing alarm first. Before touching any service, set a $1 CloudWatch billing alert. Takes 2 minutes, saves headaches.

IAM least privilege from day one. Never use root credentials for daily work. Create an IAM user immediately and lock root away.

S3 = "storage for everything." When in doubt, it's probably S3. Hosting, backups, data lakes, logs — S3 touches nearly every architecture.

Read the FAQs, not just docs. AWS service FAQs are exam gold. They answer "why would you use X over Y?" exactly like the exam does.

Draw the architecture first. Before coding any project, sketch the AWS services and how they connect. It forces clarity and reveals cost early.
